Diseño web

Un buen diseño web es aquel que considera dentro de su desarrollo tanto los elementos básicos del diseño gráfico (la diagramación, el color, los gráficos y las fuentes) como los fundamentos técnicos (estructura, compatibilidad, funcionalidad) para crear tanto el impacto visual como la experiencia de usuario óptima para la asimilación del contenido.

El diseño web debe seguir unos requerimientos mínimos de so cesibilidad web que haga que sus sitios web o aplicaciones puedan ser visitadas por el mayor número de personas.
